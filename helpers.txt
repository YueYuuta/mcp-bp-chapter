comandos
1- npm init -y
2- npm install @modelcontextprotocol/sdk zod
3- npm install --save-dev typescript ts-node @types/node
4- npx tsc --init
5- pegar en tsconfig.json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "Node",
    "esModuleInterop": true,
    "strict": true,
    "skipLibCheck": true
  }
}

6- crear archivo y pegar el primer server mcp
